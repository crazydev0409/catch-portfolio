# Catch — Time, Figured Out

Landing page for Catch, the coordination layer that connects people, businesses, and autonomous systems into a shared timeline.

## Tech Stack

- **Next.js 16** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Build

```bash
npm run build
npm start
```

## Design

- **Style**: Linear.app-inspired infrastructure software aesthetic
- **Colors**: Primary `#000000`, Accent `#A3CB31`
- **Typography**: Inter (headlines/body), JetBrains Mono (developer sections)
